Steganography Project for Image Processing Assignement, January 2019

OpenCV based project to encode any file into an image.

Build with 64bits generator

Modify OpenCV paths in CMakelists to match your computer setup.
Add opencv dlls path to environment variables

Release and Debug config are available, may need to compile two times or compile the project SteganographyLib first

Only tested on Visual Studio 15 2017

In the source code some path examples are given but they may need to be changed depending on where the .exe is
The examples will only work if the program is compiled and run from visual studio
